# JavaFX Bar Chart using Rectangle

## Description
This JavaFX program displays a bar chart using Rectangle shapes to represent grade distribution.

## Categories
- Projects: 20% (Red)
- Quizzes: 10% (Blue)
- Midterm Exams: 30% (Green)
- Final Exam: 40% (Orange)

## Features
- Bars proportional to percentage
- Labels displayed under each bar
- Horizontal layout using HBox

## How to Run
Run with JavaFX properly configured.
